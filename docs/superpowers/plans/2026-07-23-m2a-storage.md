# M2a — Storage and Saved Requests Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist collections/folders/requests to `~/.request-kit` so a request survives create → save → quit → relaunch → rename → delete with no data loss, and corrupt/duplicate files quarantine deterministically.

**Architecture:** Pure `&Path` IO helpers in `src-tauri/src/storage/` (atomic writes, slugify, scan, quarantine, node CRUD); a stateful `Workspace` (tree + `id→path` index + root) held in `AppState` behind a `Mutex`, disk being the source of truth and the index a rebuildable cache; Tauri commands orchestrate write-then-index-update under one lock. Frontend gains a `workspace` Pinia store, a sidebar tree with context-menu CRUD, and real save/open wiring into the single tab.

**Tech Stack:** Rust (serde/serde_json, uuid, tempfile, chrono), Tauri 2, Vue 3 + Pinia, Vitest.

## Global Constraints

- Storage root: `dirs::home_dir().join(".request-kit")`; never hardcode separators.
- All files JSON with top-level `"version": 1`; loader rejects `version > 1` (quarantine, never a hard failure).
- Unix permissions: dirs `0700`, files `0600`. Windows: inherited ACLs.
- Atomic writes: `NamedTempFile::new_in(parent)` → `to_writer_pretty` → `flush` → `sync_all` → `persist(target)` → best-effort parent-dir fsync on Unix. Original preserved on failure.
- Slug rules: strip `<>:"/\|?*`, trailing dots/spaces, reserved stems (`CON`,`PRN`,`AUX`,`NUL`,`COM1..9`,`LPT1..9`); empty → `untitled`; collisions get `-2`,`-3`. Case-only renames via two-step temp rename.
- Duplicate embedded UUIDs: paths normalized to `/`-separated relative form, sorted lexicographically, first wins, rest quarantined.
- No arbitrary paths cross IPC — commands take ids/names, resolve paths server-side.
- Symlinks inside app storage rejected.
- serde camelCase across the IPC boundary. All commands `Result<T, AppError>`.
- Storage mutations rejected when `AppMode != Normal` (`maintenanceInProgress`).
- Rust does not re-model the full `RequestFile`; it parses a `FileHeader { version, id, name }` and round-trips the full document as `serde_json::Value`.
- Frontend imports Tauri APIs only in `src/ipc/` and `src/main.ts`.
